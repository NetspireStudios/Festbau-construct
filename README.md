# Festbau Construction Website

A professional construction company website showcasing services and projects.

## Project Structure

```
Festbau Website/
├── index.html              # Home page
├── about.html              # About Us page
├── qualifications.html     # Qualifications page
├── projects.html           # Projects portfolio
├── contact.html            # Contact page
├── services/               # Services directory
│   ├── index.html          # Services overview
│   ├── design-build.html
│   ├── general-contracting.html
│   ├── project-management.html
│   └── modular-prefabrications.html
└── assets/                 # Static assets
    ├── css/
    │   └── style.css       # Main stylesheet
    ├── js/
    │   └── main.js         # Main JavaScript
    └── images/             # Images directory
```

## Pages

- **Home** - Landing page
- **About Us** - Company information
- **Qualifications** - Certifications and credentials
- **Services** - Service offerings with sub-pages:
  - Design Build
  - General Contracting
  - Project Management
  - Modular Prefabrications
- **Projects** - Portfolio of completed projects
- **Contact Us** - Contact information and form

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

Simply open `index.html` in a web browser to view the website.

## Development

The website is built with semantic HTML5, responsive CSS, and vanilla JavaScript. No build tools or frameworks required.
